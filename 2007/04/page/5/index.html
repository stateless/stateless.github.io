Installing Zimbra OS on Ubuntu

April 1, 2007 @ 10:14 pm · Filed under Tech

It’s pretty easy to get Open Source Zimbra running on Ubuntu 6.06 LTS base server install:


admin@mail:~/$ wget "http://downloads.sourceforge.net/zimbra/zcs-4.5.3_GA_733.UBUNTU6.tgz?modtime=1172791115&big_mirror=0"
admin@mail:~/$ tar zxf zcs-4.5.3_GA_733.UBUNTU6.tgz
admin@mail:~/$ cd zcs
admin@mail:~/zcs$ sudo apt-get install libidn11 curl fetchmail libpcre3 libgmp3c2 libexpat1 libxml2 openssl perl-modules
admin@mail:~/zcs$ ./install.sh

The Zimbra install script is quite straight forward, trickest thing is watching out for the hostname of the server vs the domain you are hosting. ie. host:mail.example.com vs @example.com. Also look out for the option to change from http only (default) mode to either both or https only for the webmail client.
Couple useful pages

  • Zmprov (Provisioning)
  • Manage Domains

Zimbra is documented quite well. In fact could be held up as an example too almost all system application howto do this well.

Update: I forgot to add that I tried to get Zimbra working last year on Ubuntu and had major issues. They have made excellent progress over this period.

Getting E61 to work with SSL for Zimbra

April 1, 2007 @ 6:01 pm · Filed under Tech

Had some trouble with getting Zimbra SSL certs to work wth my E61. This wiki page was not completely right for some reason. Using IE and getting the (tomcat) SSL cert from the web page is also important. The tomcat server seems to have a different cert from the main CA/smtp/etc services. Not sure while.

For some reason Firefox doesn’t allow you to save a SSL cert to disk. Or at least I haven’t found the correct method.

Anyway I’ve got pop to work well with Zimbra on the E61, previously I got the Zimbra Mobile to work with a trial Network Edition version. I might try out Funambol bext to see if it works with the Open Source version of Zimbra.
